## Facebook Tools v1.0
### Install
```
$ pkg update && pkg upgrade
$ pkg install python2
$ pip2 install prompt_toolkit
$ pip2 install requests bs4 cookiejar mechanize
$ pkg install git
$ git clone https://github.com/Ariya-Coder/dk-fb
$ cd dk-fb
$ python2 dark.py
```
